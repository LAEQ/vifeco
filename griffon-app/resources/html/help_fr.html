# Section Actions
gestion des vidéos Cette section vous permet de:
  • afficher les vidéos enregistrées dans la base de données
  • modifier l'usager assigné à une vidéo
  • modifier la collection assignée à une vidéo
  • afficher le total de points par catégorie pour une vidéo
  • sélectionner une vidéo pour commencer un comptage

Modifiez la collection assignée à une vidéo supprimera tous les points des catégories qui ne sont plus présentes dans la nouvelle collection.

ajout d'une vidéo

Ajouter une nouvelle vidéo dans la base de données.

La vidéo est copiée dans un répertoire spécifique de l'application.
gestion des utilisateurs Cette section vous permet de:
  • créer un nouvel utilisateur
  • éditer ses informations
  • supprimer un utilisateur

Un utilisateur 'défaut' est créé au cours de la première exécution. Cet utilisateur est indispensable pour le bon fonctionnement de l'application et ne doit pas être supprimé.

Un utilisateur ne peut pas être supprimé s'il est assigné à une vidéo.

gestion des catégories Cette section vous permet de:
  • créer une nouvelle catégorie
  • éditer une catégorie existante
  • supprimer une catégorie

Cette version ne supporte uniquement les SVG ayant qu'un seul attribut 'd'.

gestion des collections Cette section vous permet de:
  • créer une nouvelle collection
  • éditer une collection existante
  • supprimer une collection
Une collection 'défaut' est créée au cours de la première exécution. Cette collection est indispensable pour le bon fonctionnement de l'application et ne doit pas être supprimée.

Une collection ne peut pas être supprimée si elle est assignée à une vidéo.

base de données Cette section vous permet de:
  • véfifier le status de la base de données. (connection, tables, ...)
  • exporter les comptages de vidéos au format json et zip.
importer un fichier de comptage Cette section vous permet d'importer un fichier json. La structure du json est:
  • path: nom de la vidéo
  • user: (id, firstname, lastname, email, isDefault)
  • duration: en millisecondes
  • collection:
    • id
    • name
    • isDefault
    • categorySet: liste des catégories
      • id
      • name
      • icon
      • color
      • shortcut
  • pointSet: liste des points
    • id
    • x and y: coordonnées cardinales
    • startDouble (en millisecondes)
    • categoryId

L'identifiant (id) de l'utilisateur et des catégories doivent être présents dans la base de données.

Section statistique Cette section vous permet d'analyser les comptages faits par différents utilisateurs et de visualiser les différences par catégorie.
