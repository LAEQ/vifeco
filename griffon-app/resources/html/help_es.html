# Sección Acciones
Gestión de vídeo Esta sección le permite:
  • Listar los vídeos en la base de datos con una corta descripción (nombre, duración, usuario, recopilación, total de puntos...)
  • Cambiar el usuario asignado a un vídeo
  • Modificar la recopilación asignada al vídeo
  • Mostrar el número de puntos contados en total y por categoría
  • Seleccionar un vídeo para iniciar o continuar el conteo

Cambiar la recopilación asignada a un vídeo eliminará los puntos de las categorías que no están en la nueva recopilación. Exporte el vídeo antes de realizar un cambio en caso de que desee revertir su selección

Adición de un vídeo

Añadir un nuevo vídeo en el banco (base o archivo) de datos

El vídeo se copia en una carpeta creada por la aplicación .

Gestión de usuarios Esta sección le permite:
  • Crear un nuevo usuario
  • Editar las informaciones del usuario
  • Eliminar un usuario

Un usuario está creado por defecto durante la primera ejecución del programa. Este usuario es indispensable para el buen funcionamiento de la aplicación. No debe ser eliminado bajo ninguna circunstancia.

No se puede eliminar un usuario que tenga vídeos asignados. Debe reasignar los vídeos a otro usuario antes de suprimirlo.

Gestión de las categorías Esta sección le permite:
  • Crear una nueva categoría
  • Editar una categoría existente
  • Eliminar una categoría

Esta versión solo admite SVG con un atributo único d

Gestión de las recopilaciones Esta sección le permite:
  • Crear una nueva recopilación
  • Editar una recopilación existente
  • Eliminar una recopilación
Se crea una recopilación por defecto durante la primera ejecución. Esta recopilación es permanente e indispensable para el buen funcionamiento de la aplicación. No debe ser eliminada bajo ninguna circunstancia

No se puede eliminar una recopilación asociada a vídeos.

Base de datos Esta sección le permite:
  • Exportar el conteo de vídeos en el formato zip (~/vifeco/export/*.zip)
Importar un archivo de conteo Esta sección le permite importar un archivo json. La estructura del json es:
  • path: nombre del vídeo
  • user: (id, firstname, lastname, email, isDefault)
  • duration: in milliseconds
  • collection:
    • id
    • name
    • isDefault
    • categorySet: lista de categorías
      • id
      • name
      • icon
      • color
      • shortcut
  • pointSet: lista de puntos
    • id
    • x, y: coordenadas cardinales
    • startDouble (en milisegundos)
    • categoryId

El identificador (id) del usuario y las categorías deben estar presentes en la base de datos.
